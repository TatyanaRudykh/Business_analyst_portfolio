## О проекте

Данный проект посвящён оптимизации бизнес-процессов онлайн-магазина "Переделкино" - веб-платформы для дистанционных покупок товаров для ремонта и для дома с функционалом: каталог товаров, корзина, оформление заказа, онлайн-оплата. 
Заказчик обратился с пожеланием увеличить популярность онлайн-магазина (увеличить количество заказов и средний чек оформленных заказов), снизить количество отмененных заказов, а также оптимизировать внутренние бизнес-процессы, чтобы оптимизировать распределение нагрузки на сотрудников и снизить издержки на доставку. В связи с этим, в ходе выполнения проекта был проведен всесторонний анализ (как внешних, так и внутренних) бизнес-процессов.

## Задачи, поставленные перед Бизнес-аналитиком:
* Провести анализ текущего состояние бизнес-процессов, а именно:
1. Процесса оформления заказа клиентом через сайт.
2. Процесса обработки сотрудниками магазина оформленного заказа и передачи его в доставку/на выдачу.

* Описать бизнес-процессы и создать их модели.
* Провести анализ пользовательского опыта. Определить точки роста.
* Разработать предложения по улучшению бизнес-процессов.
* Провести выявление, сбор, документирование требований к решению.
* Разработать прототип решения.

## Артефакты проекта: 
* Отчёт об обследовании бизнес-процессов, содержащий AS IS процессы в нотации BPMN 2.0, CJM.
* Требования к решению: TO BE процессы в нотации BPMN 2.0, в UML (UseCases, Class Diagram, Object Diagram), описание BR, USM, Acceptance Criterias, UseCases.
* Прототип решения (Miro).
