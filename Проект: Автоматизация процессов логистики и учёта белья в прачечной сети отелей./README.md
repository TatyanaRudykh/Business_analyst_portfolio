## О проекте
Данный проект посвящен анализу и улучшению процессов логистики и учёта белья в централизованной прачечной сети отелей «Oasis Retreats».
В результате быстрого расширения сеть "Оазис" открыла 7 отелей в регионе за два года, что значительно увеличило операционную нагрузку. Существующая логистическая система перестала справляться с растущим объемом перевозок постельного белья между отелями и центральной прачечной.
Цель проекта — проанализировать текущий процесс логистики и учёта белья и спроектировать систему, которая автоматизирует планирование поставок и позволит снизить затраты сети отелей на хранение белья и пополнение его запасов, сократить время доставки белья, увеличить лояльность и удовлетворённость клиентов и сотрудников сети отелей.

## Задачи, поставленные перед Бизнес-аналитиком:
* Провести анализ текущего состояние бизнес-процессов, а именно:
1. Процесса планирования доставки.
2. Процесса доставки белья.
3. Процесса забора белья.
4. Процесса инвентаризации.
   
* Описать бизнес-процессы и создать их модели.
* Разработать предложения по улучшению бизнес-процессов.
* Описать функциональноые и нефункциональные требования к решению.
* Разработать UseCase для функциональных требований по доработке системы логистики между отелями и прачечной.
* Разработать для каждого функционального и нефункционального требования TestCase.

## Артефакты проекта: 
* Отчёт об обследовании бизнес-процессов, содержащий TO BE процессы в нотации BPMN 2.0.
* Техническое задание на доработку системы логистики и учета белья (ФТ, НФТ).
* UseCase Diagram "Процесс планирования доставки в системе "Л-логистика""
* Class Diagram, Object Diagram "Процесс планирования доставки в системе "Л-логистика""
* Требования к предложенному решению (UseCases).
* ПМИ (TestCases).
